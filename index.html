# Örnek: api/python endpoint'ine kod gönderimi (curl)
# Bu istek, API'ye "print('Hello from Python')" kodunu çalıştırmasını söyler.
# Deploy ettiğinizde: https://senin-domain.vercel.app/api/python?code=print("Hello from Python")

curl -G --data-urlencode "code=print('Hello from Python')" \
  "https://senin-domain.vercel.app/api/python"
Kullanım:
Kod parametresini URL içine gönder.
Güvenlik:
Bu demo açık API için tehlikeli olabilir. Prod ortamda doğrulama ekle.
Not:
Vercel ile python runtime kullanılmalıdır (vercel.json ayarı).
